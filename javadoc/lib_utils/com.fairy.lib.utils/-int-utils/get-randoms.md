# getRandoms

`fun getRandoms(size: Int, maxValue: Int = -1, repeat: Boolean = false): IntArray`

获取指定数量的随机数，可以指定是否允许重复

### Parameters

`size` - 指定数量,取值范围大于0

`maxValue` - 最大值，如果小于0，无限制

`repeat` - 是否允许重复，true：允许

**Return**
长度为size的随机数数组
